Gerador de planilhas

Este programa foi desenvolvido com o objetivo de automatizar a organização e centralização de informações de computadores de diferentes setores. O sistema funciona realizando a leitura de uma pasta principal que contém subpastas separadas por setores. Dentro de cada setor, existem arquivos no formato JSON com os dados individuais de cada computador.
